Fuzz-js

fuzz-js is an open source fuzzy string searching library written in JavaScript. It finds imperfect string matches inside a given array of objects.

# Live Demo

Type something into the input field below to see it in action! Select a row for some visualizations on how the match was scored.

# How scoring works

Scoring is based on "edit distance" or how many operations it takes to transform one string into the other. At the bottom of the search demo, there is a tab labeled "Fuzzalytics". When you select a row, a colored grid shows up in that tab.

The first grid is the operation matrix. The query string is displayed down the left hand side of the grid, and the subject string is displayed along the top. The cells marked "o" are the operations taken to change the query string into the subject string.

These operations have some cost associated with them. The lowest cost path path is found by filling in the edit cost grid. You can view the edit costs by clicking on the "View Scores" button.

You can read more about how edit distance works here: https://en.wikipedia.org/wiki/Levenshtein_distance

# Usage

The "filter" function filters out objects that do not pass the fuzz test.

# Advanced Usage

The "search" function returns additional search information. This includes the key of the matched string, edit distance, and a styled string based on the match ranges.

The "search" function also accepts an options object for customizing the search. To try out different options, go to the "Search Options" tab in the demo above.

If subjectKeys is not provided, the default behavior is to search all (deep) values within the object.
